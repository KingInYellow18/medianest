    Deployment Guide¶

    Complete deployment documentation for MediaNest across all environments.

    Overview¶

    MediaNest supports flexible deployment options from local development to production environments. This guide consolidates all deployment strategies, configurations, and best practices.

    Deployment Options¶

    Quick Deployment Options¶

    • Docker Compose - Single-server deployment with containers
    • Local Development - Development environment setup
    • Production Server - Traditional server deployment

    Advanced Deployment¶

    • Kubernetes - Container orchestration deployment
    • Cloud Deployment - AWS, GCP, Azure deployment
    • High Availability - Multi-server HA setup

    Specialized Deployments¶

    • Homelab Setup - Home server optimized deployment
    • Raspberry Pi - ARM-based deployment
